# Weather App

## Project Description
A simple Angular application that displays weather forecast data fetched from the Open-Meteo API. Shows date, temperature (°C and °F), and weather summary in a table, with loading and error states, a refresh button, and responsive design for desktop and mobile.

## Angular Version
Angular 21 (CLI 21.2.13)

## Installation Steps
1. Clone the repository:
   ```
   git clone https://github.com/ananyahegde/weather_app.git
   cd weather-app
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Run the app locally:
   ```
   ng serve
   ```
4. Open `http://localhost:4200` in your browser.
